# 設定・パラメータ

## Setting（基本設定）

### tcp（TCP ポート番号）

- **型:** String
- **デフォルト:** `"8080"`
- **説明:** MT5/MT4 からのシグナルを受信する TCP ポート番号

### order_type（注文タイプ）

- **型:** String
- **選択肢:**

| 値 | ラベル | 動作 |
|----|-------|------|
| `"0"` | Simple | ボラティリティ検出 → エントリー → 待機 → 決済 |
| `"1"` | Entry Buy | 買いエントリーのみ（売り方向の変動は無視） |
| `"2"` | Entry Sell | 売りエントリーのみ（買い方向の変動は無視） |
| `"3"` | Exit | 決済のみ（ボラティリティ検出時に決済クリック） |
| `"99"` | Origin | MT5/MT4 から送信される flag 値で動作を決定 |

### speed（処理速度）

- **型:** String
- **選択肢:**

| 値 | ラベル | 待機時間 |
|----|-------|---------|
| `"0"` | Ultra | 1 マイクロ秒 |
| `"1"` | High | 100 ミリ秒 |
| `"2"` | Medium | 1 秒 |
| `"3"` | Low | 3 秒 |

Ticker 受信のポーリング間隔に影響する。

### vol（ボラティリティ閾値）

- **型:** String（Decimal にパース）
- **デフォルト:** `"0.1"`
- **説明:** 価格変動がこの値を超えた場合に注文をトリガーする
- **単位:** pips またはティック（通貨ペアに依存）
- **例:** USDJPY で `"0.05"` = 0.05円 = 5銭 の変動で発動

### interval（エントリー→決済の待機時間）

- **型:** u32
- **デフォルト:** `10`
- **単位:** 秒
- **説明:** Simple モードでエントリー後、決済するまでの待機秒数

### interval_random（ランダム化）

- **型:** bool
- **デフォルト:** `true`
- **説明:** `true` の場合、interval を ±50% の範囲でランダム化する
- **例:** interval=10 の場合、5秒～15秒の間でランダムに待機

## Mouse（マウス座標設定）

3つのマウス設定が存在する:

| 設定名 | 用途 |
|--------|------|
| `mouse_entry_buy` | 買い注文ボタンの座標範囲 |
| `mouse_entry_sell` | 売り注文ボタンの座標範囲 |
| `mouse_exit` | 決済ボタンの座標範囲 |

### パラメータ

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| `start_x` | u32 | クリック範囲の左上 X 座標 (px) |
| `start_y` | u32 | クリック範囲の左上 Y 座標 (px) |
| `end_x` | u32 | クリック範囲の右下 X 座標 (px) |
| `end_y` | u32 | クリック範囲の右下 Y 座標 (px) |
| `n` | u8 | クリック回数 |

**制約:** `start_x < end_x` かつ `start_y < end_y` である必要がある。

### 座標設定の考え方

```
ブラウザ画面
┌───────────────────────────────────┐
│                                   │
│    ┌─────────┐   ┌─────────┐     │
│    │  Buy    │   │  Sell   │     │
│    │ (start) │   │         │     │
│    │  ■──────┤   │         │     │
│    │  │ 範囲 │   │         │     │
│    │  │      │   │         │     │
│    │  ├──────■   │         │     │
│    │  │  (end)│   │         │     │
│    └─────────┘   └─────────┘     │
│                                   │
│    ┌─────────────────────────┐   │
│    │       Close / Exit       │   │
│    └─────────────────────────┘   │
└───────────────────────────────────┘
```

クリックはこの矩形範囲内のランダムな座標に対して行われる。

### confirm コマンドでの座標確認

`confirm(t, n)` コマンドで実際にマウスを移動して座標を確認できる。

- `t`: 対象 (3=Buy, 4=Sell, 5=Exit)
- `n`: 移動回数

## 設定の永続化

設定は JSON ファイルとして保存・読み込みされる（`middleware/file.rs`）。

```json
{
  "tcp": "8080",
  "order_type": "0",
  "speed": "1",
  "vol": "0.1",
  "interval": 10,
  "interval_random": true
}
```

## MT5/MT4 側の Ticker JSON 仕様

TCP 経由で送信する JSON フォーマット:

```json
{
  "symbol": "USDJPY",
  "bid": "150.123",
  "ask": "150.125",
  "flag": 0,
  "side": 1,
  "server_at": "2024-01-01T12:00:00+09:00"
}
```

### flag 値（Origin モード時に使用）

| flag | 動作 |
|------|------|
| 0 | なし（無視） |
| 1 | 買いエントリーのみ |
| 2 | 売りエントリーのみ |
| 3 | 買いエントリー → 決済 |
| 4 | 売りエントリー → 決済 |
| 5 | 買い決済のみ |
| 6 | 売り決済のみ |

### side 値

| side | 意味 |
|------|------|
| 0 | なし |
| 1 | 買い |
| 2 | 売り |
